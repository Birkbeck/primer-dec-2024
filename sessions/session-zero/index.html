Introduction — Session Zero!

  • Learning objectives

    By the end of this session, you will have an understanding of

    • An overview of this course, our remote tooling, and the Department.
    • A (very) brief intro to Computer Science with "a bit" of history.
    • What the edit-compile-run cycle is, what are editors, and how helpful is an Integrated Development Environment (IDE)?
    • How computers work (straightforward overview - something like the materials you will see in the Computer Systems module of the CS programme).
  • Pre-session activities

    We will use the CODIO virtual coding environment for our programming sessions; you should create an account at codio. co.uk to gain access to the materials. Following the enrollment link should allow you to join the course.
